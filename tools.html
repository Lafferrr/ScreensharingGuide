Forensic Tools Repository

The complete collection of tools, parsers, scripts, and download links for comprehensive PC checking.

GUI Analyzers & External Tools (.exe / .jar)

System Informer

Kernel-mode process manager for thread inspection, memory string analysis, and Kernel Live Dumps.

Detect It Easy (DiE)

Static analysis for executables — entropy, packer detection, string extraction.

Hayabusa

Fast forensic timeline generator and threat-hunting tool for Windows Event Logs.

Luyten

Java decompiler — converts .class bytecode back into readable source code.

Recaf

Java bytecode editor — decompile, inspect, and analyze .jar files.

RedLotusAltChecker

Automated GUI tool for scanning launcher configs to find Alt accounts.

MeowDoomsdayFucker (Tonynoh)

Specific detection tool for traces of the DoomsDay cheat client.

BAMReveal (Orbdiff)

Extended BAM data viewer with enhanced analysis capabilities.

Echo Tools (.exe)

Echo Journal Tool

Parses and filters USN Journal logs efficiently via GUI.

Echo String Tool

Scans multiple custom strings simultaneously against a selected process.

Echo UserAssist View

GUI viewer for Windows UserAssist execution logs.

Echo USBDEVIEW

USB history viewer showing connect/disconnect timestamps.

Eric Zimmerman Tools (.exe)

bstrings

Advanced string extraction and regex filtering from files and memory dumps.

PECmd

Command-line Prefetch file parser with full loaded-file output.

RECmd (Registry Explorer)

Parses Windows Registry hives for offline analysis.

SSTools (Lafferr)

Lily's Collector

AcousticVoids Screenshare

Enr1c0o All-in-one

Mod & Cheat Specific Scanners

MeowModAnalyzer (Tonynoh)

Analyzes the active Minecraft JVM process — arguments, classpath, and active mod list.

YarpModAnalyzer (YarpLetapStan)

Alternative mod analyzer with extended detection signatures.

MeowImportsChecker (Tonynoh)

Scans C:\ for files/folders and analyzes strings for external cheat and macro client patterns.

JAR Scanner (Lily)

Detects hash identifiers, solid obfuscation, java agents, and native code in .jar files.

JARParser (orbdiff)

Extracts executed .jar file traces from Prefetch and DcomLaunch strings.

Faker Detections (Lily)

Detects PC swap bypasses via WLAN, ARP, and MAC analysis.

System Trace Parsers (BAM, Prefetch, USN)

Spokwn BAM Parser

Parses BAM registry key for executed file paths, timestamps, and digital signatures.

BAMReveal (Orbdiff)

Extended BAM data viewer with additional analysis capabilities.

Prefetch Integrity (bacanoicua)

Checks for missing or read-only Prefetch files indicating log tampering.

Services / Uptime (Lily)

Checks critical services, prefetch state, event log state, and recycle bin.

Suspicious Scheduler (nolww)

Parses scheduled tasks and flags unsigned apps masquerading as system tasks.

Hard Link Finder (Lily)

Scans the MFT for hardlinks modified in the current boot or last hour.

Common Directories (Lily)

Scans System32, SysWOW64, and %temp% for non-Microsoft-signed files.

CMD Import Scripts (Run in CMD or CMD as Admin)

All of the following are run directly in CMD — not PowerShell. Copy and paste into a CMD window.

Service Checker (Lily)

Checks all critical services, prefetch state, event logs, and recycle bin. 
powershell Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass && powershell Invoke-Expression (Invoke-RestMethod https://raw.githubusercontent.com/praiselily/lilith-ps/refs/heads/main/Services.ps1)

MeowModAnalyzer (Tonynoh)

Analyzes the running Minecraft JVM — arguments, classpath, active mods. 
powershell -ExecutionPolicy Bypass -Command "Invoke-Expression (Invoke-RestMethod 'https://raw.githubusercontent.com/MeowTonynoh/MeowModAnalyzer/main/MeowModAnalyzer.ps1')"

YarpModAnalyzer (YarpLetapStan)

Alternative mod analyzer with different detection signatures. 
powershell -ExecutionPolicy Bypass -Command "Invoke-Expression (Invoke-RestMethod https://raw.githubusercontent.com/YarpLetapStan/PowershellScripts/refs/heads/main/YarpsModAnalyzer6.0.ps1)"

SSTools Collector (Lafferr)

Downloads the full tool suite to C:\SSTools. 
powershell -ExecutionPolicy Bypass -Command "Invoke-Expression (Invoke-WebRequest -Uri 'https://raw.githubusercontent.com/Lafferrr/SSTools/main/SSTools.ps1' -UseBasicParsing).Content"
