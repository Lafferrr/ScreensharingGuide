Alt Account Detection

Methods for finding hidden alternative accounts across various launchers and platforms.

Automated Detection

The fastest way to scan for alternative accounts is to use the dedicated RedLotusAltChecker. It automates the parsing of configuration files and extracts linked Minecraft and Discord accounts across all major launchers.

https://github.com/ItzIceHere/RedLotusAltChecker/releases/download/RL/RedLotusAltChecker.exe


Alternatively, use Lily's GUI Alt Detector for a comprehensive scan across browsers, discord canary, and .gz files:

https://github.com/praiselily/AltDetector

Manual JSON Inspection

If automated tools are unavailable or blocked, you can manually inspect the JSON configuration files of popular Minecraft launchers. Look for the accounts.json or equivalent files which store authentication tokens and usernames.

Vanilla / General

%AppData%\.minecraft\launcher_profiles.json %AppData%\.minecraft\launcher_accounts.json

Lunar Client

%UserProfile%\.lunarclient\settings\game\accounts.json

LabyMod

%AppData%\.minecraft\LabyMod\accounts.json

Prism Launcher

%AppData%\PrismLauncher\accounts.json

MultiMC

%AppData%\MultiMC\accounts.json

TLauncher

%AppData%\.tlauncher\accounts.json
Important: Never copy or steal a user's session token or UUID. Your objective is only to verify the account names tied to the launcher. Stealing tokens is a severe privacy violation and illegal.
